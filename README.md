# basic-port-scanner
Multi-threaded TCP port scanner with banner grabbing and service detection.

## Features
- Multi-threaded scanning (fast performance)
- Banner grabbing (identify software versions)
- Service detection (HTTP, SSH, MySQL, etc.)
- Two scan modes: Common ports or Full range (1-65535)

## Usage
```bash
python scanner.py
```

Enter target IP and choose scan type when prompted.

## Requirements
- Python 3.x
- No external libraries needed

## Legal Notice
**Only scan systems you own or have explicit permission to test.**  
Unauthorized port scanning may be illegal in your jurisdiction.
